\doxysubsection{Detailed Description}
High-\/level HTTP client with connection pooling and protocol negotiation. 

High-\/level HTTP client abstracting HTTP/1.\+1 and HTTP/2 with\+:
\begin{DoxyItemize}
\item Connection pooling with per-\/host limits
\item Automatic protocol negotiation (ALPN for HTTP/2)
\item Cookie handling (RFC 6265)
\item Authentication (Basic/\+Digest/\+Bearer)
\item Compression (gzip/deflate/brotli)
\item Redirect following
\item Both synchronous and asynchronous APIs
\end{DoxyItemize}

Dependencies (leveraged, not duplicated)\+:
\begin{DoxyItemize}
\item Socket\+HTTP for headers, URI, methods, status codes
\item Socket\+HTTP1 for HTTP/1.\+1 parsing/serialization
\item Socket\+HTTP2 for HTTP/2 protocol
\item Socket\+Happy\+Eyeballs for fast connection establishment
\item Socket\+DNS for asynchronous DNS resolution
\item Socket\+Crypto for authentication (base64, MD5, SHA256)
\end{DoxyItemize}

Thread safety\+: Client instances are NOT thread-\/safe. Use one client per thread or external synchronization.

PLATFORM REQUIREMENTS\+:
\begin{DoxyItemize}
\item POSIX-\/compliant system (Linux, BSD, mac\+OS)
\item pthread for mutex synchronization
\item Open\+SSL for TLS (optional, via SOCKET\+\_\+\+HAS\+\_\+\+TLS)
\end{DoxyItemize}
