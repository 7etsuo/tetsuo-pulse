\doxysubsection{Detailed Description}
Protocol-\/agnostic HTTP types, header handling, URI parsing, and utilities. 

Provides protocol-\/agnostic HTTP types, header handling, URI parsing, and date/media type utilities. Foundation for HTTP/1.\+1 and HTTP/2.

Features\+:
\begin{DoxyItemize}
\item HTTP methods with semantic properties (safe, idempotent, cacheable)
\item HTTP status codes with reason phrases and categories
\item Header collection with O(1) case-\/insensitive lookup
\item RFC 3986 URI parsing with percent-\/encoding support and syntax validation
\item HTTP-\/date parsing (all 3 formats per RFC 9110)
\item Media type parsing with token validation and escape handling
\item Content negotiation (Accept header q-\/value parsing)
\end{DoxyItemize}

Thread safety\+: All functions are thread-\/safe (no global state). Header collections are not thread-\/safe; use external synchronization if sharing across threads.

Security notes\+:
\begin{DoxyItemize}
\item Rejects control characters and invalid syntax in URI components
\item Validates host as reg-\/name or basic IPv6 literal
\item Media types/parameters validated as HTTP tokens (RFC 7230)
\item Per-\/component length limits prevent resource exhaustion
\item Header names/values validated to reject injection attacks
\item Integer overflow protection on all size calculations
\end{DoxyItemize}
