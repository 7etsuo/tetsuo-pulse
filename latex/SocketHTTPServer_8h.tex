\doxysubsection{Detailed Description}
High-\/level HTTP server supporting HTTP/1.\+1 and HTTP/2 protocols. 

High-\/level HTTP server supporting HTTP/1.\+1 and HTTP/2\+:
\begin{DoxyItemize}
\item Event-\/driven request handling
\item Connection management with keep-\/alive
\item Protocol negotiation (ALPN for HTTP/2)
\item Web\+Socket upgrade support
\item Request body streaming for large uploads
\item Response body streaming (chunked transfer encoding)
\item HTTP/2 server push support
\item Rate limiting per endpoint
\item Per-\/client connection limiting
\item Request validation middleware
\item Granular timeout enforcement
\item Graceful shutdown (drain)
\end{DoxyItemize}

Dependencies (leveraged, not duplicated)\+:
\begin{DoxyItemize}
\item Socket\+HTTP for headers, URI, methods, status codes
\item Socket\+HTTP1 for HTTP/1.\+1 parsing
\item Socket\+HTTP2 for HTTP/2 protocol
\item Socket\+Poll for event loop integration
\item Socket\+Rate\+Limit for rate limiting
\item Socket\+IPTracker for per-\/client limits
\end{DoxyItemize}

Thread safety\+: Server instances are NOT thread-\/safe. Use one server per thread or external synchronization.

Platform Requirements\+:
\begin{DoxyItemize}
\item POSIX-\/compliant system (Linux, BSD, mac\+OS)
\item pthread for mutex synchronization
\item Open\+SSL for TLS (optional, via SOCKET\+\_\+\+HAS\+\_\+\+TLS)
\end{DoxyItemize}
