\doxysubsection{Detailed Description}
Configuration constants for HTTP client with compile-\/time override support. 

Centralized configuration for HTTP client module. All magic numbers are defined here with compile-\/time override support.

Configurable Limits Summary\+:

All limits can be overridden at compile time with -\/D flags or at runtime via \doxylink{structSocketHTTPClient__Config}{Socket\+HTTPClient\+\_\+\+Config} fields.

Resource Limits\+:
\begin{DoxyItemize}
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+MAX\+\_\+\+RESPONSE\+\_\+\+SIZE\+: 0 (unlimited) -\/ Max response body size
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+MAX\+\_\+\+CONNS\+\_\+\+PER\+\_\+\+HOST\+: 6 -\/ Per-\/host connection limit
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+MAX\+\_\+\+TOTAL\+\_\+\+CONNS\+: 100 -\/ Total connection limit
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+MAX\+\_\+\+REDIRECTS\+: 10 -\/ Max redirect hops
\item HTTPCLIENT\+\_\+\+MAX\+\_\+\+AUTH\+\_\+\+RETRIES\+: 2 -\/ Max auth retry attempts
\end{DoxyItemize}

Timeout Limits\+:
\begin{DoxyItemize}
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+CONNECT\+\_\+\+TIMEOUT\+\_\+\+MS\+: 30s -\/ Connection establishment timeout
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+REQUEST\+\_\+\+TIMEOUT\+\_\+\+MS\+: 60s -\/ Full request completion timeout
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+DNS\+\_\+\+TIMEOUT\+\_\+\+MS\+: 10s -\/ DNS resolution timeout
\item HTTPCLIENT\+\_\+\+DEFAULT\+\_\+\+IDLE\+\_\+\+TIMEOUT\+\_\+\+MS\+: 60s -\/ Idle connection timeout
\end{DoxyItemize}

Enforcement\+:
\begin{DoxyItemize}
\item max\+\_\+response\+\_\+size\+: Checked during body accumulation (raises Response\+Too\+Large)
\item max\+\_\+conns\+\_\+per\+\_\+host\+: Enforced by connection pool
\item max\+\_\+redirects\+: Checked before each redirect (raises Too\+Many\+Redirects)
\end{DoxyItemize}

Metrics\+:
\begin{DoxyItemize}
\item SOCKET\+\_\+\+CTR\+\_\+\+LIMIT\+\_\+\+RESPONSE\+\_\+\+SIZE\+\_\+\+EXCEEDED incremented on size violation
\end{DoxyItemize}
