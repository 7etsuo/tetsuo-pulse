\doxysubsection{Detailed Description}
Internal HTTP/2 connection and stream structures. 

This header contains internal structures for the HTTP/2 protocol implementation. NOT for public consumption -\/ use \doxylink{SocketHTTP2_8h}{Socket\+HTTP2.\+h} instead.

Contains\+:
\begin{DoxyItemize}
\item HTTP/2 connection state machine and internal structures
\item Stream lifecycle management and state tracking
\item Frame processing and flow control internals
\item Rate limiting for RST\+\_\+\+STREAM and PING flood protection
\item Connection preface and settings management
\item HPACK integration for header compression
\end{DoxyItemize}

The implementation follows RFC 9113 with security hardening for DoS protection.
