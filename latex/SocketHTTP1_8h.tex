HTTP/1.\+1 message syntax parsing and serialization (RFC 9112). 

Provides HTTP/1.\+1 message parsing, serialization, and chunked encoding. Builds on \doxylink{SocketHTTP_8h}{Socket\+HTTP.\+h} for core types (methods, status codes, headers, URI).

Features\+:
\begin{DoxyItemize}
\item DFA-\/based incremental parser (O(n) complexity)
\item Request and response parsing
\item All request-\/target forms (origin, absolute, authority, asterisk)
\item Chunked transfer encoding with trailer support
\item Request smuggling prevention (strict RFC 9112 Section 6.\+3)
\item Optional content encoding (gzip/deflate/brotli)
\item Configurable limits for security
\end{DoxyItemize}

Thread safety\+: Parser instances are NOT thread-\/safe. Use one parser per thread or external synchronization.

Security notes\+:
\begin{DoxyItemize}
\item Rejects requests with both Content-\/\+Length and Transfer-\/\+Encoding
\item Rejects multiple differing Content-\/\+Length values
\item Validates all header names/values for injection attacks
\item Enforces configurable size limits
\end{DoxyItemize}
