\doxysubsection{Detailed Description}
HTTP/2 protocol implementation (RFC 9113) with multiplexing and flow control. 

Provides complete HTTP/2 implementation with\+:
\begin{DoxyItemize}
\item Binary framing layer (9-\/byte frame headers)
\item Stream multiplexing with state machine
\item Flow control (connection and stream level)
\item HPACK header compression integration
\item Server push support
\item h2c cleartext upgrade support
\end{DoxyItemize}

Dependencies\+:
\begin{DoxyItemize}
\item Socket\+HPACK for header compression
\item Socket\+HTTP for HTTP semantics
\item Socket\+Buf for I/O buffering
\item Socket\+\_\+T for underlying transport
\item Arena for memory management
\end{DoxyItemize}

Thread safety\+: Connection instances are NOT thread-\/safe. Use one connection per thread or external synchronization.

Security notes\+:
\begin{DoxyItemize}
\item Enforces MAX\+\_\+\+CONCURRENT\+\_\+\+STREAMS to prevent resource exhaustion
\item HPACK bomb prevention via Socket\+HPACK limits
\item Flow control window overflow prevention
\item SETTINGS\+\_\+\+TIMEOUT enforcement
\end{DoxyItemize}
