\doxysubsection{Detailed Description}
Internal HTTP server structures and connection management. 

This header contains internal structures for the HTTP server implementation. NOT for public consumption -\/ use \doxylink{SocketHTTPServer_8h}{Socket\+HTTPServer.\+h} instead.

Contains\+:
\begin{DoxyItemize}
\item Server connection lifecycle and state management
\item Request/response processing state machines
\item Rate limiting per endpoint and per-\/client limits
\item Connection pooling and resource tracking
\item Request body streaming and response streaming
\item Web\+Socket upgrade support internals
\item HTTP/2 server push capability
\end{DoxyItemize}

The server supports both HTTP/1.\+1 and HTTP/2 with graceful shutdown and comprehensive resource limits for DoS protection.
