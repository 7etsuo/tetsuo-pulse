\doxysubsection{Detailed Description}
Internal HTTP/1.\+1 parser structures and DFA state machine. 

This header contains internal structures for the HTTP/1.\+1 parser implementation. NOT for public consumption -\/ use \doxylink{SocketHTTP1_8h}{Socket\+HTTP1.\+h} instead.

Contains\+:
\begin{DoxyItemize}
\item Table-\/driven DFA character classification (following Socket\+UTF8 pattern)
\item HTTP/1.\+1 state machine definitions and transition tables
\item Parser internal structures and buffers
\item Chunked encoding state machine
\item Compression/decompression state (if enabled)
\end{DoxyItemize}

The parser uses a deterministic finite automaton (DFA) for O(n) parsing with single-\/pass validation and error detection.
