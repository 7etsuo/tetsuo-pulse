\doxysubsection{Detailed Description}
HPACK header compression (RFC 7541) for HTTP/2. 

Provides HPACK header compression for HTTP/2, including\+:
\begin{DoxyItemize}
\item Static table (61 common headers per RFC 7541 Appendix A)
\item Dynamic table with FIFO eviction
\item Huffman encoding/decoding (RFC 7541 Appendix B)
\item Variable-\/length integer encoding (RFC 7541 Section 5.\+1)
\end{DoxyItemize}

Features\+:
\begin{DoxyItemize}
\item O(n) single-\/pass Huffman decoding via DFA
\item O(1) dynamic table add/evict via circular buffer
\item HPACK bomb prevention with configurable limits
\item Thread-\/safe exception handling
\end{DoxyItemize}

Thread safety\+: Encoder/decoder instances are NOT thread-\/safe. Use one instance per thread or external synchronization.

Security notes\+:
\begin{DoxyItemize}
\item Enforces maximum header size limits
\item Validates Huffman padding (max 7 bits of 1s)
\item Limits dynamic table size updates per header block
\item Never indexes sensitive headers marked with never\+\_\+index
\end{DoxyItemize}
