46/* ============================================================================
47 * Configuration Limits
48 * ============================================================================
49 *
50 * CONFIGURABLE LIMITS SUMMARY
51 *
52 * All limits can be overridden at compile time with -D flags or at runtime
53 * via SocketHTTP1_Config.
54 *
55 * MESSAGE STRUCTURE LIMITS:
56 * SOCKETHTTP1_MAX_REQUEST_LINE - 8KB - Max request/status line
57 * SOCKETHTTP1_MAX_URI_LEN - 8KB - Max URI length
58 * SOCKETHTTP1_MAX_HEADER_NAME - 256B - Max header name length
59 * SOCKETHTTP1_MAX_HEADER_VALUE - 8KB - Max header value length
60 * SOCKETHTTP1_MAX_HEADERS - 100 - Max number of headers
61 * SOCKETHTTP1_MAX_HEADER_SIZE - 64KB - Max total header size
62 *
63 * CHUNKED ENCODING LIMITS:
64 * SOCKETHTTP1_MAX_CHUNK_SIZE - 16MB - Max single chunk size
65 * SOCKETHTTP1_MAX_CHUNK_EXT - 1KB - Max chunk extension length
66 * SOCKETHTTP1_MAX_TRAILER_SIZE - 4KB - Max trailer headers size
67 *
68 * ENFORCEMENT:
69 * - All limits enforced during parsing
70 * - Returns HTTP1_ERROR_LINE_TOO_LONG, HTTP1_ERROR_HEADER_TOO_LARGE, etc.
71 * - Request smuggling prevention per RFC 9112 Section 6.3
72 *
73 * SECURITY:
74 * - Rejects both Content-Length AND Transfer-Encoding (smuggling prevention)
75 * - Rejects multiple differing Content-Length values
76 * - Validates all header names/values for injection attacks
77 */
