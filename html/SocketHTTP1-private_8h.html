Detailed Description

Internal HTTP/1.1 parser structures and DFA state machine.

This header contains internal structures for the HTTP/1.1 parser implementation. NOT for public consumption - use SocketHTTP1.h instead.

Contains:

  • Table-driven DFA character classification (following SocketUTF8 pattern)
  • HTTP/1.1 state machine definitions and transition tables
  • Parser internal structures and buffers
  • Chunked encoding state machine
  • Compression/decompression state (if enabled)

The parser uses a deterministic finite automaton (DFA) for O(n) parsing with single-pass validation and error detection.

See also
SocketHTTP1.h for public HTTP/1.1 API.
SocketUTF8.c for similar table-driven DFA pattern.
SocketHTTP-private.h for core HTTP internal structures.
