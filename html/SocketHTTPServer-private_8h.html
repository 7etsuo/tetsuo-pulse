Detailed Description

Internal HTTP server structures and connection management.

This header contains internal structures for the HTTP server implementation. NOT for public consumption - use SocketHTTPServer.h instead.

Contains:

  • Server connection lifecycle and state management
  • Request/response processing state machines
  • Rate limiting per endpoint and per-client limits
  • Connection pooling and resource tracking
  • Request body streaming and response streaming
  • WebSocket upgrade support internals
  • HTTP/2 server push capability

The server supports both HTTP/1.1 and HTTP/2 with graceful shutdown and comprehensive resource limits for DoS protection.

See also
SocketHTTPServer.h for public HTTP server API.
SocketHTTP-private.h for core HTTP internal structures.
SocketIPTracker.h for per-client connection limiting.
SocketRateLimit.h for rate limiting integration.
