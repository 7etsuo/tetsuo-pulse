Detailed Description

Internal HTTP/2 connection and stream structures.

This header contains internal structures for the HTTP/2 protocol implementation. NOT for public consumption - use SocketHTTP2.h instead.

Contains:

  • HTTP/2 connection state machine and internal structures
  • Stream lifecycle management and state tracking
  • Frame processing and flow control internals
  • Rate limiting for RST_STREAM and PING flood protection
  • Connection preface and settings management
  • HPACK integration for header compression

The implementation follows RFC 9113 with security hardening for DoS protection.

See also
SocketHTTP2.h for public HTTP/2 API.
SocketHPACK.h for header compression integration.
SocketHTTP-private.h for core HTTP internal structures.
