Detailed Description

Complete HTTP/1.1 and HTTP/2 protocol implementation with client and server support.

The HTTP group provides comprehensive HTTP protocol support including parsing, serialization, client/server implementations, and advanced features. Key components include:

  • SocketHTTP (core): HTTP types, headers, URI parsing, status codes
  • SocketHTTP1 (http1): HTTP/1.1 parsing and serialization
  • SocketHTTP2 (http2): HTTP/2 protocol implementation
  • SocketHTTPClient (client): High-level HTTP client with pooling
  • SocketHTTPServer (server): HTTP server implementation
  • SocketHPACK (hpack): HTTP/2 header compression
See also
foundation for base infrastructure.
core_io for socket primitives.
security for TLS integration.
SocketHTTPClient_T for HTTP client usage.
SocketHTTPServer_T for HTTP server implementation.
