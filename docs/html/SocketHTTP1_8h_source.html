39/* ============================================================================
40 * Configuration Limits
41 * ============================================================================
42 *
43 * CONFIGURABLE LIMITS SUMMARY
44 *
45 * All limits can be overridden at compile time with -D flags or at runtime
46 * via SocketHTTP1_Config.
47 *
48 * MESSAGE STRUCTURE LIMITS:
49 * SOCKETHTTP1_MAX_REQUEST_LINE - 8KB - Max request/status line
50 * SOCKETHTTP1_MAX_URI_LEN - 8KB - Max URI length
51 * SOCKETHTTP1_MAX_HEADER_NAME - 256B - Max header name length
52 * SOCKETHTTP1_MAX_HEADER_VALUE - 8KB - Max header value length
53 * SOCKETHTTP1_MAX_HEADERS - 100 - Max number of headers
54 * SOCKETHTTP1_MAX_HEADER_SIZE - 64KB - Max total header size
55 *
56 * CHUNKED ENCODING LIMITS:
57 * SOCKETHTTP1_MAX_CHUNK_SIZE - 16MB - Max single chunk size
58 * SOCKETHTTP1_MAX_CHUNK_EXT - 1KB - Max chunk extension length
59 * SOCKETHTTP1_MAX_TRAILER_SIZE - 4KB - Max trailer headers size
60 *
61 * ENFORCEMENT:
62 * - All limits enforced during parsing
63 * - Returns HTTP1_ERROR_LINE_TOO_LONG, HTTP1_ERROR_HEADER_TOO_LARGE, etc.
64 * - Request smuggling prevention per RFC 9112 Section 6.3
65 *
66 * SECURITY:
67 * - Rejects both Content-Length AND Transfer-Encoding (smuggling prevention)
68 * - Rejects multiple differing Content-Length values
69 * - Validates all header names/values for injection attacks
70 */
