88/* ============================================================================
89 * Configuration Limits
90 * ============================================================================
91 *
92 * CONFIGURABLE LIMITS SUMMARY
93 *
94 * All limits can be overridden at compile time with -D flags or at runtime
95 * via SocketHTTP1_Config.
96 *
97 * MESSAGE STRUCTURE LIMITS:
98 * SOCKETHTTP1_MAX_REQUEST_LINE - 8KB - Max request/status line
99 * SOCKETHTTP1_MAX_URI_LEN - 8KB - Max URI length
100 * SOCKETHTTP1_MAX_HEADER_NAME - 256B - Max header name length
101 * SOCKETHTTP1_MAX_HEADER_VALUE - 8KB - Max header value length
102 * SOCKETHTTP1_MAX_HEADERS - 100 - Max number of headers
103 * SOCKETHTTP1_MAX_HEADER_SIZE - 64KB - Max total header size
104 *
105 * CHUNKED ENCODING LIMITS:
106 * SOCKETHTTP1_MAX_CHUNK_SIZE - 16MB - Max single chunk size
107 * SOCKETHTTP1_MAX_CHUNK_EXT - 1KB - Max chunk extension length
108 * SOCKETHTTP1_MAX_TRAILER_SIZE - 4KB - Max trailer headers size
109 *
110 * ENFORCEMENT:
111 * - All limits enforced during parsing
112 * - Returns HTTP1_ERROR_LINE_TOO_LONG, HTTP1_ERROR_HEADER_TOO_LARGE, etc.
113 * - Request smuggling prevention per RFC 9112 Section 6.3
114 *
115 * SECURITY:
116 * - Rejects both Content-Length AND Transfer-Encoding (smuggling prevention)
117 * - Rejects multiple differing Content-Length values
118 * - Validates all header names/values for injection attacks
119 */
