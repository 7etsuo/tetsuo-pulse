89/* ============================================================================
90 * Configuration Limits
91 * ============================================================================
92 *
93 * CONFIGURABLE LIMITS SUMMARY
94 *
95 * All limits can be overridden at compile time with -D flags or at runtime
96 * via SocketHTTP1_Config.
97 *
98 * MESSAGE STRUCTURE LIMITS:
99 * SOCKETHTTP1_MAX_REQUEST_LINE - 8KB - Max request/status line
100 * SOCKETHTTP1_MAX_URI_LEN - 8KB - Max URI length
101 * SOCKETHTTP1_MAX_HEADER_NAME - 256B - Max header name length
102 * SOCKETHTTP1_MAX_HEADER_VALUE - 8KB - Max header value length
103 * SOCKETHTTP1_MAX_HEADERS - 100 - Max number of headers
104 * SOCKETHTTP1_MAX_HEADER_SIZE - 64KB - Max total header size
105 *
106 * CHUNKED ENCODING LIMITS:
107 * SOCKETHTTP1_MAX_CHUNK_SIZE - 16MB - Max single chunk size
108 * SOCKETHTTP1_MAX_CHUNK_EXT - 1KB - Max chunk extension length
109 * SOCKETHTTP1_MAX_TRAILER_SIZE - 4KB - Max trailer headers size
110 *
111 * ENFORCEMENT:
112 * - All limits enforced during parsing
113 * - Returns HTTP1_ERROR_LINE_TOO_LONG, HTTP1_ERROR_HEADER_TOO_LARGE, etc.
114 * - Request smuggling prevention per RFC 9112 Section 6.3
115 *
116 * SECURITY:
117 * - Rejects both Content-Length AND Transfer-Encoding (smuggling prevention)
118 * - Rejects multiple differing Content-Length values
119 * - Validates all header names/values for injection attacks
120 */
