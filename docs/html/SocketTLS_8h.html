Detailed Description

TLS/SSL socket integration with TLS 1.3 support.

Features:

  • TLS1.3-only (strict PFS, no legacy vulns)
  • Transparent I/O integration
  • Non-blocking handshake with poll
  • SNI + hostname verification
  • ALPN support
See also
SocketTLS_enable() for enabling TLS on existing sockets.
SocketTLSContext_T for TLS context management.
SocketDTLS_T for DTLS support on UDP sockets.
