websocket_server.c - WebSocket Echo Server Example

Demonstrates a WebSocket server that echoes messages back to clients. Integrates with HTTP server for WebSocket upgrade handling.

Build: cmake -DBUILD_EXAMPLES=ON .. make example_websocket_server

Usage: ./example_websocket_server [port] ./example_websocket_server 8080

Test with browser console: ws = new WebSocket('ws://localhost:8080/ws'); ws.onmessage = (e) => console.log('Received:', e.data); ws.send('Hello, Server!');

Or with the websocket_client example: ./example_websocket_client localhost 8080 /ws
