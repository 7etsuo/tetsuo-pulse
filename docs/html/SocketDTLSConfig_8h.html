SocketDTLSConfig.h - DTLS Configuration Constants.

Defines secure defaults for DTLS operations: protocol versions, MTU settings, cookie protection parameters, timeouts, and limits. Provides stub typedefs when TLS disabled for compilation without OpenSSL.

All constants can be overridden before including this header. Enforces DTLS 1.2 minimum for security (DTLS 1.3 when OpenSSL 3.2+ widely available).

Thread-safe: Yes - compile-time constants

References:

  • RFC 6347: Datagram Transport Layer Security Version 1.2
  • RFC 9147: The Datagram Transport Layer Security (DTLS) Protocol Version 1.3
