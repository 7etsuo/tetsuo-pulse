Detailed Description

DTLS configuration constants and secure defaults.

Defines secure defaults for DTLS operations: protocol versions, MTU settings, cookie protection parameters, timeouts, and limits. Provides stub definitions when TLS disabled for compilation without OpenSSL.

All constants can be overridden before including this header. Enforces DTLS 1.2 minimum for security (DTLS 1.3 when OpenSSL 3.2+ widely available).
