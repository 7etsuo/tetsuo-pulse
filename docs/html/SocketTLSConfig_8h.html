struct SocketTLSConfig_T

SocketTLSConfig.h - TLS Configuration Constants and Stubs.

Defines secure defaults for TLS operations: TLS1.3-only, modern ciphers, buffer sizes, timeouts, and limits. Provides stub typedefs when TLS disabled for compilation without OpenSSL.

All constants can be overridden before including this header. Enforces high-security posture by default (no legacy protocols/ciphers).

Thread-safe: Yes - compile-time constants
