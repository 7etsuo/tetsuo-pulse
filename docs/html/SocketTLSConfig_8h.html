Detailed Description

TLS configuration constants, structure, and secure defaults.

Defines secure defaults for TLS operations: TLS1.3-only protocols, modern cipher suites, buffer sizes, timeouts, limits, and configuration structure (SocketTLSConfig_T). Provides stub typedefs when TLS is disabled for compilation without OpenSSL/LibreSSL. Includes initialization function SocketTLS_config_defaults() for the config struct.

All constants can be overridden before including this header to customize security parameters. Enforces high-security posture by default: no legacy protocols/ciphers, strict version pinning.

Quick Start Example

#include "tls/SocketTLSConfig.h"
#include "tls/SocketTLSContext.h"
#include "socket/Socket.h"
// Initialize secure config
SocketTLSConfig_T cfg;
SocketTLS_config_defaults(&cfg);
// Create context
SocketTLSContext_T ctx = SocketTLSContext_new(&cfg);
// Secure a socket
TRY {
Socket_T sock = Socket_new(AF_INET, SOCK_STREAM, 0);
Socket_connect(sock, "example.com", 443);
SocketTLS_enable(sock, ctx);
// ... perform TLS handshake and I/O ...
Socket_free(&sock);
} EXCEPT(SocketTLS_Failed) {
// Handle TLS errors
} END_TRY;
SocketTLSContext_free(&ctx);
