Detailed Description

TLS internal shared definitions and helper functions.

Internal header for all TLS module implementation files. Contains shared helper function declarations, error handling macros, internal types, and the SocketTLSContext_T structure definition. NOT part of public API - do not include from application code.

Thread safety: Internal functions are not thread-safe unless noted.

See also
SocketTLS.h for public TLS API.
SocketTLSContext.h for public TLS context API.
