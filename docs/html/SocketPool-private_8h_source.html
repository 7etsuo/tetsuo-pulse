176 * Thread-local exception for detailed error messages across all SocketPool
177 * implementation files. Uses the centralized error buffer (socket_error_buf)
178 * from SocketUtil.h for consistent error formatting.

180 * Benefits:
181 * - Single thread-local error buffer (socket_error_buf) for all modules
182 * - Consistent error formatting with SOCKET_ERROR_FMT/MSG macros
183 * - Thread-safe exception raising
184 * - Automatic logging integration via SocketLog_emit
185 *
186 * NOTE: For multi-file modules like SocketPool, we use an extern declaration
187 * here and the actual definition in SocketPool-core.c. This allows all
188 * implementation files to share the same thread-local exception variable.
