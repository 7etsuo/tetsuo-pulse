105 * @brief Thread-local exception for detailed error messages.
106 * @ingroup connection_mgmt
107 *
108 * Thread-local exception for detailed error messages across all SocketPool
109 * implementation files. Uses the centralized error buffer (socket_error_buf)
110 * from SocketUtil.h for consistent error formatting.
111 *
112 * Benefits:
113 * - Single thread-local error buffer (socket_error_buf) for all modules
114 * - Consistent error formatting with SOCKET_ERROR_FMT/MSG macros
115 * - Thread-safe exception raising
116 * - Automatic logging integration via SocketLog_emit
117 *
118 * NOTE: For multi-file modules like SocketPool, we use an extern declaration
119 * here and the actual definition in SocketPool-core.c. This allows all
120 * implementation files to share the same thread-local exception variable.
121 */
