Detailed Description

Secure default constants for DTLS protocol parameters, MTU management, cookie protection, timeouts, and DoS protection limits.

These constants can be overridden before including this header and provide stubs when TLS is disabled. Enforces DTLS 1.2 minimum with support for 1.3 when available.

See also
SocketDTLS.h for the DTLS socket API.
SocketDTLSContext.h for context configuration.
