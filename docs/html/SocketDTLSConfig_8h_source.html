57/* ============================================================================
58 * MTU and Buffer Sizes
59 * ============================================================================
60 * DTLS requires careful MTU management to avoid IP fragmentation
61 * which can cause packet loss and performance degradation.
62 *
63 * Path MTU discovery is recommended but not always reliable over UDP.
64 * Conservative defaults ensure interoperability.
65 */

98/* ============================================================================
