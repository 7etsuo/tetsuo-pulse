Detailed Description

Internal implementation details for WebSocket module.

Contains private structures, constants, and helper functions for SocketWS. Not for public use - API unstable and may change without notice.

References:

  • RFC 6455: The WebSocket Protocol
  • RFC 7692: Compression Extensions for WebSocket (permessage-deflate)
