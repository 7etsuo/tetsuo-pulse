Detailed Description

DTLS internal shared definitions and helper functions.

Internal header for all DTLS module implementation files. Contains shared helper function declarations, error handling macros, internal types, and the SocketDTLSContext_T structure definition. NOT part of public API - do not include from application code.

Thread safety: Internal functions are not thread-safe unless noted.

See also
SocketDTLS.h for public DTLS API.
SocketDTLSContext.h for public DTLS context API.
