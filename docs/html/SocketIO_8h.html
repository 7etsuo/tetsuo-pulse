Detailed Description

Internal I/O abstraction layer for socket operations with TLS support.

Provides internal I/O operations that automatically route through TLS when enabled, or use raw socket operations otherwise. This abstraction layer handles the complexity of TLS/non-TLS operation selection.

See also
Socket_send() for public send operations.
Socket_recv() for public receive operations.
SocketTLS_T for enabling TLS on sockets.
SocketAsync_T for async I/O integration.
SocketPoll_T for event-driven I/O.
