Detailed Description

Internal SYN flood protection implementation details for the Security Modules group.

Private header containing internal structures and functions for the SocketSYNProtect module. Not part of the public API - do not include directly from user code.

Contains:

  • IP reputation tracking hash tables
  • Sliding window counters for rate measurement
  • CIDR range parsing and prefix matching management
  • Score calculation and decay algorithms
  • Whitelist/blacklist data structures
Note
Do not include directly - use SocketSYNProtect.h instead.
Warning
Internal implementation details subject to change.
See also
SocketSYNProtect.h for public SYN protection API.
