61/* ============================================================================
62 * Configuration Constants
63 * ============================================================================
64 *
65 * CONFIGURABLE LIMITS SUMMARY
66 *
67 * All limits can be overridden at compile time with -D flags or at runtime
68 * via SocketHTTPServer_Config fields.
69 *
70 * RESOURCE LIMITS:
71 * HTTPSERVER_DEFAULT_MAX_HEADER_SIZE - 64KB - Max total header size
72 * HTTPSERVER_DEFAULT_MAX_BODY_SIZE - 10MB - Max request body size
73 * HTTPSERVER_DEFAULT_MAX_CONNECTIONS - 1000 - Max concurrent connections
74 * HTTPSERVER_DEFAULT_MAX_CONNECTIONS_PER_CLIENT - 100 - Per-IP limit
75 * HTTPSERVER_DEFAULT_MAX_REQUESTS_PER_CONN - 1000 - Max requests per connection
76 * HTTPSERVER_DEFAULT_MAX_CONCURRENT_REQUESTS - 100 - HTTP/2 streams
77 *
78 * TIMEOUT LIMITS:
79 * HTTPSERVER_DEFAULT_REQUEST_TIMEOUT_MS - 30s - Idle timeout
80 * HTTPSERVER_DEFAULT_KEEPALIVE_TIMEOUT_MS - 60s - Keep-alive timeout
81 * HTTPSERVER_DEFAULT_REQUEST_READ_TIMEOUT_MS - 30s - Full request read
82 * HTTPSERVER_DEFAULT_RESPONSE_WRITE_TIMEOUT_MS - 60s - Full response write
83 *
84 * ENFORCEMENT:
85 * - max_header_size: Enforced by HTTP/1.1 parser (returns error)
86 * - max_body_size: Enforced before body allocation (returns 413)
87 * - max_connections: Enforced in accept loop (rejects new clients)
88 * - max_connections_per_client: Enforced via SocketIPTracker
89 *
90 * METRICS:
91 * - SOCKET_CTR_LIMIT_BODY_SIZE_EXCEEDED incremented on body limit violation
92 * - SOCKET_CTR_LIMIT_HEADER_SIZE_EXCEEDED incremented on header limit violation
93 */
