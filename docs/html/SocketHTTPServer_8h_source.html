68 * Configuration Constants
69 * ============================================================================
70 *
71 * CONFIGURABLE LIMITS SUMMARY
72 *
73 * All limits can be overridden at compile time with -D flags or at runtime
74 * via SocketHTTPServer_Config fields.
75 *
76 * RESOURCE LIMITS:
77 * HTTPSERVER_DEFAULT_MAX_HEADER_SIZE - 64KB - Max total header size
78 * HTTPSERVER_DEFAULT_MAX_BODY_SIZE - 10MB - Max request body size
79 * HTTPSERVER_DEFAULT_MAX_CONNECTIONS - 1000 - Max concurrent connections
80 * HTTPSERVER_DEFAULT_MAX_CONNECTIONS_PER_CLIENT - 100 - Per-IP limit
81 * HTTPSERVER_DEFAULT_MAX_REQUESTS_PER_CONN - 1000 - Max requests per
82 * connection HTTPSERVER_DEFAULT_MAX_CONCURRENT_REQUESTS - 100 - HTTP/2 streams
83 *
84 * TIMEOUT LIMITS:
85 * HTTPSERVER_DEFAULT_REQUEST_TIMEOUT_MS - 30s - Idle timeout
86 * HTTPSERVER_DEFAULT_KEEPALIVE_TIMEOUT_MS - 60s - Keep-alive timeout
87 * HTTPSERVER_DEFAULT_REQUEST_READ_TIMEOUT_MS - 30s - Full request read
88 * HTTPSERVER_DEFAULT_RESPONSE_WRITE_TIMEOUT_MS - 60s - Full response write
89 *
90 * ENFORCEMENT:
91 * - max_header_size: Enforced by HTTP/1.1 parser (returns error)
92 * - max_body_size: Enforced before body allocation (returns 413)
93 * - max_connections: Enforced in accept loop (rejects new clients)
94 * - max_connections_per_client: Enforced via SocketIPTracker
95 *
96 * METRICS:
97 * - SOCKET_CTR_LIMIT_BODY_SIZE_EXCEEDED incremented on body limit violation
98 * - SOCKET_CTR_LIMIT_HEADER_SIZE_EXCEEDED incremented on header limit
99 * violation
100 */
