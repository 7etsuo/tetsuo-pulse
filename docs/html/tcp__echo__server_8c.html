tcp_echo_server.c - Basic TCP Echo Server Example

Demonstrates a simple TCP echo server using the core Socket API. Shows basic socket operations: bind, listen, accept, recv, send.

Build: cmake -DBUILD_EXAMPLES=ON .. make example_tcp_echo_server

Usage: ./example_tcp_echo_server [port] ./example_tcp_echo_server 8080

Test with: echo "Hello, World!" | nc localhost 8080 telnet localhost 8080
