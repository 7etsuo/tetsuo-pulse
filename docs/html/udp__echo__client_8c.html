udp_echo_client.c - Basic UDP Echo Client Example

Demonstrates a simple UDP echo client using the SocketDgram API. Shows UDP socket operations: connect, send, recv.

Build: cmake -DBUILD_EXAMPLES=ON .. make example_udp_echo_client

Usage: ./example_udp_echo_client [host] [port] [message] ./example_udp_echo_client localhost 8080 "Hello, World!"
