Detailed Description

Defines internal backend abstraction for cross-platform polling.

Not part of public API - include SocketPoll.h for application use. Backend implementors must provide epoll/kqueue/poll specific code in src/poll/.

Note
Internal header; subject to change without notice.
Warning
Backends assume non-threaded access; do not use concurrently.
