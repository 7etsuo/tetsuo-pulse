http_server.c - Basic HTTP Server Example

Demonstrates a simple HTTP server using the SocketHTTPServer API. Supports HTTP/1.1 with keep-alive.

Build: cmake -DBUILD_EXAMPLES=ON .. make example_http_server

Usage: ./example_http_server [port] ./example_http_server 8080

Test with: curl http://localhost:8080/ curl http://localhost:8080/hello curl http://localhost:8080/json curl -X POST -d "test=data" http://localhost:8080/echo
