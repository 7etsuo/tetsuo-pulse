Detailed Description

HTTP/2 (RFC 9113) support with framing, streams, flow control, and HPACK integration.

Complete HTTP/2 implementation including binary framing, stream multiplexing, flow control, HPACK header compression, server push, and h2c upgrade.

Key features:

  • Full RFC 9113 compliance
  • Client and server support
  • Automatic preface handling and SETTINGS exchange
  • Stream lifecycle management
  • Connection and stream flow control
  • DoS protections (rate limits, frame size validation)
  • Integration with SocketHTTP for semantics and SocketHPACK for compression
See also
SocketHTTP for protocol-agnostic HTTP types.
SocketHPACK for HPACK header compression.
SocketHTTP1 for HTTP/1.1 upgrade path.
SocketHTTPClient and SocketHTTPServer for high-level usage.

Complete HTTP/2 connection preface and initial settings exchange.

Advances the connection through the initial HTTP/2 handshake state machine (RFC 9113 Section 3.4-3.5). This includes sending/receiving the connection preface, exchanging SETTINGS frames, and acknowledging them. Call repeatedly until returns 0 (complete).

State transitions:

  • CLIENT from INIT: Send preface magic + initial SETTINGS, advance to PREFACE_SENT
