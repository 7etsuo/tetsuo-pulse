Detailed Description

Internal structures and state for Happy Eyeballs connection racing (RFC 8305).

Part of the Socket Library.

This header contains internal implementation details for the Happy Eyeballs module. Not for public use - structures and functions may change without notice.

The Happy Eyeballs algorithm races IPv6 and IPv4 connection attempts to minimize connection latency on dual-stack hosts.

See also
SocketHappyEyeballs.h for the public API.
Async I/O module for integration with event loops.
SocketDNS.h for asynchronous DNS resolution used in racing.
