http2_client.c - HTTP/2 Client Example

Demonstrates HTTP/2 with ALPN negotiation and protocol detection. The SocketHTTPClient automatically negotiates HTTP/2 when available.

Build: cmake -DBUILD_EXAMPLES=ON .. make example_http2_client

Usage: ./example_http2_client [url] ./example_http2_client https://www.google.com

Note: HTTP/2 requires TLS (HTTPS). HTTP/2 cleartext (h2c) is also supported but rarely used in practice.
