Detailed Description

Per-request builder and state for custom HTTP requests.

HTTP request builder.

Holds method, URI, headers, body data or stream callback, timeouts, and authentication overrides for a single request. Arena-allocated for temporary use.

See also
SocketHTTPClient_Request_new() for creation.
SocketHTTPClient_Request_execute() to send the request.
