Detailed Description

Internal timer implementation details and heap management.

Private header containing internal structures and functions for the SocketTimer module. Not part of the public API - do not include directly from user code.

Contains:

  • Timer heap implementation with binary heap operations
  • Timer ID generation and management
  • Internal timer structure definitions
Note
Thread-safe where noted - heap operations use mutex protection.
See also
SocketTimer.h for public timer API.
