600/* ============================================================================
601 * Graceful Shutdown (Drain) API
602 * ============================================================================
603 *
604 * Industry-standard graceful shutdown following patterns from nginx, HAProxy,
605 * and Go http.Server. Provides clean state machine transitions, non-blocking
606 * APIs for event loop integration, and timeout-guaranteed completion.
607 *
608 * State Machine:
609 * drain(timeout)
610 * +---------+ ───────────────> +----------+
611 * | RUNNING | | DRAINING |
612 * +---------+ +----------+
613 * ^ │
614 * │ │ (count == 0 OR timeout)
615 * │ +----------+ │
616 * +────────────| STOPPED |<─────────+
617 * (restart) +----------+
618 *
619 * Typical usage:
620 * SocketPool_drain(pool, 30000); // Start 30s drain
621 * while (SocketPool_drain_poll(pool) > 0) {
622 * // Continue event loop, connections closing naturally
623 * SocketPoll_wait(poll, &events, SocketPool_drain_remaining_ms(pool));
624 * }
625 * SocketPool_free(&pool);
626 */
