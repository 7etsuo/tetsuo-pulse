596/* ============================================================================
597 * Graceful Shutdown (Drain) API
598 * ============================================================================
599 *
600 * Industry-standard graceful shutdown following patterns from nginx, HAProxy,
601 * and Go http.Server. Provides clean state machine transitions, non-blocking
602 * APIs for event loop integration, and timeout-guaranteed completion.
603 *
604 * State Machine:
605 * drain(timeout)
606 * +---------+ ───────────────> +----------+
607 * | RUNNING | | DRAINING |
608 * +---------+ +----------+
609 * ^ │
610 * │ │ (count == 0 OR timeout)
611 * │ +----------+ │
612 * +────────────| STOPPED |<─────────+
613 * (restart) +----------+
614 *
615 * Typical usage:
616 * SocketPool_drain(pool, 30000); // Start 30s drain
617 * while (SocketPool_drain_poll(pool) > 0) {
618 * // Continue event loop, connections closing naturally
619 * SocketPoll_wait(poll, &events, SocketPool_drain_remaining_ms(pool));
620 * }
621 * SocketPool_free(&pool);
622 */
